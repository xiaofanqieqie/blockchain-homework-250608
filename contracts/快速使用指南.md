# 🎯 众筹合约交互式测试工具 - 快速使用指南

## 🚀 启动工具

```bash
cd contracts
npx hardhat run scripts/interactive-test.js --network localhost
```

## 🎮 基本操作流程

### 1️⃣ 创建项目
选择 `1` → 输入项目信息
```
项目标题: 智能手表众筹
项目描述: 一款具有健康监测功能的智能手表  
目标金额: 1.5
众筹天数: 30
```

### 2️⃣ 查看项目
选择 `6` → 留空查看所有项目 或 输入项目ID查看详情

### 3️⃣ 投资项目
选择 `2` → 输入项目ID → 输入投资金额(ETH)

### 4️⃣ 提取资金 (项目成功后)
选择 `3` → 输入项目ID → 确认提取

### 5️⃣ 查看账户信息
选择 `13` → 查看余额和活动统计

## 💡 快速测试

**完整测试流程:**
1. 启动工具 → 选择 `1` 创建项目 (目标1ETH)
2. 选择 `2` 投资 0.6ETH
3. 选择 `2` 再投资 0.5ETH (项目成功!)
4. 选择 `3` 提取资金 (扣除2.5%手续费)
5. 选择 `6` 查看项目状态

## ⚙️ 可用功能

- ✅ 创建/查看项目 (1,6)
- ✅ 投资/提取资金 (2,3)  
- ✅ 账户/网络信息 (13,14)
- 🔧 其他功能开发中 (4,5,7-12)

## 🚪 退出程序
选择 `0` 或按 `Ctrl+C`

---
**提示**: 每次操作后按回车继续，所有交易都会显示Gas使用量和确认状态。
